# Contributing to Text-2-Video

Thank you for your interest! This project welcomes contributions from developers of all levels.

## How to Contribute
1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature`.
3. Make changes and add tests.
4. Run `python run_t2v.py` to verify.
5. Submit a PR with a clear description.

## Guidelines
- Follow PEP 8 for Python code.
- Add docstrings to new functions.
- Update README for new features.
- Ensure GPU compatibility.

## Code of Conduct
Be respectful and inclusive. Report issues via GitHub.
